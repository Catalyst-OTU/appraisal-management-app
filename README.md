## Environment Configure 
- Installing virtual Env
    - pip install pipenv 

- Installing Packages
    - pipenv install 

- Starting Virtual Env
    - pipenv shell 

- Running FastAPI Service Locally 
    - uvicorn app.main:app --reload

- Running FastAPI Service On Docker 
    - Start Docker Service 
    - docker-compose build
    - docker-compose up -d
